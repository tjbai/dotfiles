# Linear

Use the Linear MCP tools as the source of truth for Linear workspace state.

Common workflows include:

- creating and updating issues, projects, and project documents
- reading project state, milestones, cycles, and status updates
- managing comments, labels, customers, and related Linear metadata

Prefer Linear MCP tools over assumptions or stale local notes.
